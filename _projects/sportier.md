**Sportier** is a mobile sports communication and player-matching application designed to connect local sports enthusiasts. It was developed during my tenure as President of the UOS Entrepreneurship Club.

### Key Accomplishments

- **Location-Based Matching**: Designed and implemented a location-based matching system that connects local players based on geographical proximity and sport preference.
- **Competitive Tier System**: Created a fair, skill-based classification system for user profiles and matchmaking.
- **Leadership**: Led the project as the club president, coordinating frontend/backend development and UI/UX design.

### Tech Stack & Tools

- **Frontend**: JavaScript, React JS
